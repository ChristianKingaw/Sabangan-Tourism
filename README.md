# Sabangan Tourism Site (Next.js)

## Run locally
1. Install dependencies: `pnpm install`
2. Start dev server: `pnpm dev`
3. Open `http://localhost:3000`

## Build for production
1. `pnpm build`
2. `pnpm start`

## What is included
- Next.js app router scaffold
- Interactive Leaflet map page at `/` using `leaflet` + `leaflet-omnivore`
- KML source available at `public/Gagayam Trail Run.kml`
- Existing static tourism page still available at `public/index.html`
